# GreenLoop - Eco-Friendly Packaging Management System

## Overview

GreenLoop is a desktop-based inventory and order management system developed for an eco-friendly packaging supply company. The application replaces manual record-keeping methods by providing a centralized platform to manage products, customers, inventory, orders, and sales efficiently.

## Features

* User Login and Authentication
* Product Management (Add, Update, Delete, Search)
* Customer Management
* Inventory and Stock Tracking
* Order Management
* Sales Management
* Report Generation
* Database Integration using JDBC and MySQL
* User-Friendly Java Swing Interface

## Technologies Used

* Java
* Java Swing
* JDBC
* MySQL
* Maven
* IntelliJ IDEA

## Project Structure

* `src/` - Java source code
* `pom.xml` - Maven dependencies and configuration
* Database scripts and resources

## Installation

1. Clone or download the project.
2. Create the required MySQL database.
3. Configure database connection settings.
4. Open the project in IntelliJ IDEA.
5. Install Maven dependencies.
6. Run the application.

## Objective

The objective of GreenLoop is to improve operational efficiency, reduce manual errors, and provide a reliable digital solution for managing eco-friendly packaging products and business transactions.
